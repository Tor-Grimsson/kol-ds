# Icon inventory QA — 885 icons vs the 24-grid (2026-07-07)

Method: every icon rendered live on `/icons` (stroke variant), measured in-browser — `getBBox()` geometry + stroke-width attributes, normalized to the 24-grid. Companion to audit finding #24 (`2026-07-07-showcase-audit.md`). This is the ledger for kol-loader normalization work; redraws are design work, deliberately not attempted here.

## Baseline — the good news

- **All 885 icons are on a square 24×24 viewBox.** No grid conversion needed anywhere.
- Only 2 truly full-bleed, 0 non-square.

## Headline — stroke weight forks three ways

| Stroke @24 | Count | Reading |
| --- | --- | --- |
| **1.5** | 556 (63%) | the de-facto KOL standard |
| **2** | 252 (28%) | second population — reads chunky next to 1.5 (`al-1`, `al-2`, check-sets…) |
| **1.125** (1.12497) | 32 | machine-scaled import (phosphor-style set: pennib, pencilsimple, alignbottom…) — reads anemic |
| none (filled shapes) | 35 | legit for solid glyphs, but includes stroke-drawn-as-outline conversions (edit-1/2/3, menu-1/2/3, settings…) |
| oddballs | 10 | mixed 1.5+2 (5), 1.5+6 (3), 2.5 (1), 0.99997 (1) |

**Recommendation: normalize on 1.5 @24.** The 2-weight population (252) is the visible inconsistency in the grid; the 1.125 set (32) is a mechanical `×0.75` scale artifact fixable by attribute rewrite (1.125 → 1.5), no redraw.

## Size / position buckets

- **Undersized (max dim < 14.5/24): 67.** Two families: (a) legitimately small glyphs — chevrons/carets/tr-* transport glyphs at 10–12 — these need a *size-class convention* (declare them "compact glyphs", don't inflate); (b) genuinely underscaled drawings (arrow-from-* at 14, boolean-merge/unite 13, rack dither/spiral tiles, shapes-15 at 8.7) — these want re-fitting to the ~18–20 keyline.
- **Off-center (centroid >1.5 off 12,12): 23.** Some optical (bulb/coffee/game hang low; volume anchors left) — judge per-icon; boolean-merge/unite (10,14) are plainly mis-registered.
- **Full-bleed: 2** — `shapes/shape-3`, `user/social-github` (24.0 — touches the frame; github is a brand mark, likely fine).
- **Thick: 4** — rack/cap-* at 3.75 (spec-diagram glyphs, arguably intentional), `hash-italic-bold` 2.5.

Full name lists for every bucket are reproducible by re-running the measurement (the collection script lives in the session transcript; 40-name samples were captured per bucket and match the counts above).
