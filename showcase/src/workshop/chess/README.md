# Styleguide · Chess Components

Scratchpad for apparatus panels, dashboard cards, and tables supporting the chess analytics program.

| Directory | Purpose |
|-----------|---------|
| `apparatus/` | Board renderer experiments, interaction controls, animation studies. |
| `dashboards/` | KPI cards, charts, and storytelling panels. |
| `tables/` | Data table prototypes for match logs, openings, and opponents. |

When components graduate, promote them into `@kol/ui/src/chess` for cross-app reuse.
