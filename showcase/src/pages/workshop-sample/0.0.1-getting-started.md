# Getting Started

Welcome to the **workshop docs system** — a live dogfood of `@kolkrabbi/kol-workshop`. The shell, the handrolled markdown engine, and the docs viewer all render real KOL chrome here.

## Installation

Feed the package your own docs:

- Add the dependency
- Supply your `import.meta.glob` module map
- Pass `routes` and `docHref`

## Owns vs reuses

| Part | Owns | Reuses |
|---|---|---|
| Engine | parser, search, tags | — |
| Shell | composition | AppShell, ShellHeader |
| Viewer | reader, article | DocsToc, CodeBlock |
