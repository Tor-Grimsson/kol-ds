# Components — the full inventory

Every public export of `@kolkrabbi/kol-component` and `@kolkrabbi/kol-framework`, by tier. Three places to see any of them live:

- **Showcase** — `/components/<slug>` (demo + code + install + mined usage + API table)
- **Workbench** — every component × every state (`pnpm workbench`)
- **Mined usage** — [`../07-usage/<Name>.md`](../07-usage/) — real, attributed examples from ~25 consumer apps (generated by `scripts/extract-usage.mjs`; regenerate after consumer apps change)

Tier rules (what makes an atom an atom) live in [[02-placement|component placement]] and are machine-enforced by `pnpm validate:taxonomy`.
